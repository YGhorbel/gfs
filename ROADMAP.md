## Future Considerations

Beyond v1.0, we're exploring:

- **Home made file system**: A tiered file system levering object storage
- **More AI Agents capabilities**: Better integrations with AI agents
- **Multi-compute**: Multi computes working on different branches
- **Merge & Rebase**: Merge and rebase of branches
